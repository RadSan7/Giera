# Projekt AI - Giera

To jest repozytorium naszego projektu.

## Jak zacząć w 3 krokach?

1. **Pobierz repozytorium** (jeśli jeszcze nie masz):
   ```bash
   git clone <link-do-repo>
   cd Giera
   ```

2. **Stwórz wirtualne środowisko** (żeby nie śmiecić w systemie):
   ```bash
   # Tworzymy środowisko o nazwie 'venv'
   python3 -m venv venv
   
   # Aktywujemy je:
   # Na Mac/Linux:
   source venv/bin/activate
   # Na Windows:
   # venv\Scripts\activate
   ```

3. **Zainstaluj biblioteki**:
   ```bash
   pip install -r requirements.txt
   ```

## Uruchamianie

Aby uruchomić projekt, wpisz:
```bash
python main.py
```
